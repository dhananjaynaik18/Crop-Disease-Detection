 1. Introduction

Agriculture plays a very important role in our country, and crop health directly affects farmers’ income and food production. Many times, farmers are not able to identify diseases in crops at an early stage, which leads to heavy losses.

This project aims to build an intelligent system that can detect crop diseases using image processing and Artificial Intelligence. By simply uploading an image of a leaf, the system can predict whether the plant is healthy or affected by a disease and also suggest possible treatments.


 2. Problem Statement

In real-life situations, farmers face the following problems:
	•	Lack of knowledge about different plant diseases
	•	Delay in identifying diseases
	•	Dependency on experts or agricultural officers
	•	Improper treatment due to wrong diagnosis

These issues reduce crop productivity and increase losses.


 3. Proposed Solution / Approach

To solve this problem, I developed a Crop Disease Detection System using Deep Learning.

 Approach Used:
	1.	Dataset Collection
I used a plant disease dataset containing images of:
	•	Tomato (Healthy & Late Blight)
	•	Potato (Healthy & Early Blight)
	2.	Image Processing
Images are resized to a fixed size (128x128) for consistent input.
	3.	Model Building (CNN)
A Convolutional Neural Network (CNN) is used because it is very effective for image classification tasks.
	4.	Training the Model
	•	Data is split into training and validation sets
	•	Model learns patterns from images
	•	Accuracy and loss are tracked
	5.	Prediction System (App)
Using Streamlit, a user-friendly interface is created where:
	•	User uploads an image
	•	Model predicts disease
	•	Confidence score is shown
	•	Treatment suggestions are provided



 4. Features of the Project
	•	 Multi-crop support (Tomato & Potato)
	•	 Deep Learning based prediction
	•	 Accuracy & Loss graph visualization
	•	 Treatment suggestions for diseases
	•	 Prediction history saved in CSV file
	•	 Interactive UI using Streamlit
	•	 Loading animation for better experience



 5. Key Design Decisions

While building this project, I made some important decisions:
	•	Used CNN model instead of traditional ML because images require feature extraction
	•	Added data augmentation to improve model accuracy
	•	Used Early Stopping to avoid overfitting
	•	Selected Streamlit for simple and fast UI development
	•	Reduced dataset to fewer classes (Tomato & Potato) for better performance on my system
